Curso JS para desarrolladores de backend

Este curso esta enfocado la adquisición de habilidades en javascript para programadores con conocimientos de backend. El backend que usaremos está hecho en Django, pero el foco no está en el backend sino en como utilizar JS para sacarle provecho a este backend desde el frontend.

Pre-requisitos

  • Conocimientos básicos de JavaScript
  • Experiencia con git
  • Experiencia con jQuery
  • Bonus: conocer de python/django así podemos explicar tópicos en términos que todos entendemos.

Objetivos

Al terminar el curso sabremos acerca detalles en el uso de JS, y estaremos más capacitados para asumir tareas de frontend.

Horario

El curso (versión español) se ofrecerá los sábados a las 12 PM (GMT+1) y cada sesión dura 1.5 horas, es absolutamente online. Por favor tomar nota el horario es (GMT+1) Nota: eventualmente se ofrecerán ayudantías en otros horarios.

Cupos

Los cupos son limitados y ocuparemos distintos criterios para llenar las vacantes.
